# 2.4 Basic Input/Output Operations

## Key Concepts

- Input
- Output
- Print Function
- Input Function

## Explanations

Input/output operations are used to interact with the user and display output in Python. The print() function is used to display output to the screen, while the input() function is used to get input from the user. The print() function can be used to display strings, numbers, and other data types, while the input() function returns a string by default.

## Examples

- print('Hello, world!')  # use the print() function to display a greeting message
- name = input('What is your name? ')  # use the input() function to get a string from the user
- age = int(input('How old are you? '))  # use the input() function to get an integer from the user

## Summary Points

- Input/output operations are used to interact with the user and display output in Python
- The print() function is used to display output to the screen
- The input() function is used to get input from the user
- The input() function returns a string by default, so you may need to convert it to another data type
- The print() function and input() function can be used together to create interactive programs

## Self-Assessment Questions

A: The print() function is used to display output to the screen.

Q: How do you get input from the user in Python?
A: You can use the input() function to get input from the user.

Q: What data type does the input() function return by default?
